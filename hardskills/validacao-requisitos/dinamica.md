# Validação de Requisitos

## Identificação
- ID: DH-004
- Tipo: Hard Skill
- Versão: 1.0

## Descrição
Dinâmica para análise crítica e validação de requisitos previamente documentados.

## Objetivos
- Identificar ambiguidades
- Detectar inconsistências
- Melhorar qualidade dos requisitos

## Público-Alvo
Estudantes de Engenharia de Software.

## Tempo de Execução
45 minutos

## Nível de Complexidade
Médio

## Passo a Passo
1. Entregar documento de requisitos.
2. Grupo identifica problemas.
3. Apresentar correções.
